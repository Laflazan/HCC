# ⚖️ HCC Law Firm Website

Modern, premium ve SEO odaklı bir hukuk bürosu web sitesi projesidir.
Bu proje, **kurumsal kimliği güçlü, sade ve güven veren** bir dijital deneyim sunmak amacıyla geliştirilmiştir.

---

## 🚀 Proje Hakkında

HCC Avukatlık Bürosu için geliştirilen bu web sitesi:

* Modern UI/UX prensiplerine uygun
* Hızlı, optimize ve SEO dostu
* Responsive (mobil + tablet + desktop uyumlu)
* Kurumsal ve premium tasarım diline sahip

---

## 🎨 Tasarım Yaklaşımı

* 🎯 Minimal & temiz arayüz
* ⚫ Dark Theme → Siyah + Altın (#C8A96A)
* ⚪ Light Theme → Beyaz + Altın
* ✨ Yumuşak animasyonlar ve geçişler
* 🧩 Simetrik ve profesyonel layout

Referans siteler:
